# Python Person Greeter 👋
A simple Python program that greets the user based on the name they enter.

---

## 📌 About the Project
This is a beginner-friendly Python project that demonstrates:
- Functions
- User input handling
- Conditional statements
- Basic input validation

---

## 🚀 Features
- Takes user name as input
- Checks if input is empty
- Displays a personalized greeting
- Handles missing input gracefully

---

## 🧠 How It Works
1. The program asks the user to enter their name
2. It checks whether the input is empty or not
3. If valid → prints a greeting message
4. If empty → shows a warning message

---

## 🛠️ Technologies Used
- Python 3

---

## ▶️ How to Run

1. Open terminal
2. Run the program:

```bash
python main.py
```
